# 🛒 Grocery Chatbot

An AI-powered chatbot that allows users to search grocery products using natural language queries.

## 🚀 Features

* AI chatbot for grocery queries
* Machine learning intent detection
* Flask API backend
* TensorFlow trained model
* Grocery database integration

## 🧠 Technologies Used

* Python
* Flask
* TensorFlow
* JSON
* Machine Learning

## 📂 Project Structure

```
grocery_chatbot/
│
├── app_flask.py
├── app.py
├── classes.json
├── grocery_database.json
├── training_history.json
├── grocery_saved_model/
├── requirements.txt
└── README.md
```

## ▶️ Run Locally

Install dependencies:

pip install -r requirements.txt

Run the chatbot:

python app_flask.py
